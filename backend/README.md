## Getting Started

This shows step by step procedures needed to get the backend up and running.

### Prerequisites
```sh
npm i
```
## Usage

In order to get the server running,
run the following:
```sh
npm start
```
The server should be running locally on port 5000.
https://www.localhost:5000
